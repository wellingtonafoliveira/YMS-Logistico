# YMS - Sistema de Gestão Logística

Sistema desenvolvido para controle de:

- Agenda de cargas
- Separação
- Expedição
- Pátio
- Docas
- Check-in de motoristas

## 🚀 Tecnologias

- HTML / CSS / JavaScript
- Supabase (Banco + Auth + Realtime)

## 📊 Funcionalidades

- Dashboard em tempo real
- Controle por status operacional
- Check-in de motoristas
- Controle de docas
- Importação via Excel

## 📌 Estrutura

- /css → estilos
- /js → lógica
- /modules → funcionalidades do sistema
